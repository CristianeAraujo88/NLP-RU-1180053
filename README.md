Criação de modelo de classificação supervisionado para análise de Fake News.
# Classificação de Fake News com NLP

Este projeto foi desenvolvido como atividade prática da disciplina de Processamento de Linguagem Natural.

## Tecnologias utilizadas
- Python
- NLTK
- Scikit-learn
- Pandas
- TF-IDF

## Modelo
Classificador Logistic Regression com acurácia de 88,22%.

## Dataset
Fake.br Corpus
